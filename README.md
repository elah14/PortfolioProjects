À propos de ce projet

Ce projet, c'est un tour d'horizon des données mondiales sur la COVID-19. Le but est de sortir des chiffres importants concernant la pandémie, d'analyser les taux de décès et d'infection, et de suivre comment avancent les campagnes de vaccination dans le monde entier.

 Compétences SQL
 
Nettoyage des données : Changer les types de données (ALTER, CAST) et gérer les données manquantes. 

Regroupements et calculs : SUM, MAX, GROUP BY pour résumer les infos par pays et continents. 

Jointures : Croiser les tables de décès et de vaccinations. 

CTE (Common Table Expressions) : Organiser les requêtes compliquées pour que ce soit plus clair. 

Fonctions de fenêtrage : OVER(PARTITION BY...) pour calculer des totaux glissants. 

Vues : Créer des vues pour balancer facilement les données vers des outils de visualisation (Power BI / Tableau). 

 Analyses Importantes
 
Ce script aide à répondre à ces questions :

Mortalité : Quel est le pourcentage de décès par rapport au nombre de cas (spécial Maroc 🇲🇦 et France 🇫🇷) ?

Infection : Quel pourcentage de la population a chopé le virus dans chaque pays ?

Classement : Quels sont les continents et les pays les plus touchés niveau nombre total de décès ?
